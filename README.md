## Essential Git Commands for DevOps

- `git init` - Initialize a new repository
- `git clone` - Clone an existing repository
- `git add` - Stage changes for commit
- `git commit` - Save changes with a message
- `git push` - Upload commits to remote repository
- `git pull` - Fetch and merge remote changes
- `git branch` - Create, list, or delete branches
- `git checkout` - Switch branches or restore files
- `git merge` - Combine branches
- `git rebase` - Reapply commits on top of another branch
- `git tag` - Create release tags
- `git log` - View commit history
- `git diff` - Show changes between commits
- `git status` - Display working tree status
- `git remote` - Manage remote repositories
- `git stash` - Temporarily save uncommitted changes
- `git reset` - Undo commits or unstage files
- `git fetch` - Download objects from remote repository
- `git config` - Set Git configuration
- `git blame` - Show who changed each line
